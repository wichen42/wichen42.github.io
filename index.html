CloudTunes is a SoundCloud clone, a music streaming and social media web app where users are able to upload their own audio files with the functionality to like, comment, and follow other users and their music.

Technologies utilized includes React and Redux for frontend, Ruby on Rails backend utilizing a PostgreSQL database management system for backend, and AWS S3 cloud storage.
